# Hashcat 密碼破解實驗儀表板

這是一個基於 React.js 開發的互動式儀表板，用於展示 Hashcat 密碼破解實驗的設計方案和數據結果分析。

## 🎨 特色

- **黑色主題設計** - 專業的黑色背景配色，提供優雅的視覺體驗
- **互動式數據可視化** - 使用 Recharts 呈現實驗數據和趨勢
- **響應式佈局** - 完美支援桌面和移動設備
- **多頁面導航** - 清晰的導航結構，輕鬆瀏覽不同內容

## 📊 主要功能

### 1. 儀表板總覽
- 實驗統計數據概覽
- 系統配置信息
- 攻擊模式統計
- 實驗階段時間軸

### 2. 實驗設計
- 實驗目標與範圍說明
- 測試方法詳細解析
- 系統配置與參數說明
- 攻擊模式與遮罩字符集說明

### 3. 數據結果分析
- **密碼長度分析** - 破解時間與密碼長度的關係
- **複雜度分析** - 不同字符類型對破解時間的影響
- **效能比較** - 各種攻擊模式的效能對比
- **總結報告** - 關鍵發現與最佳實踐建議

## 🚀 快速開始

### 安裝依賴

```bash
cd exam/dashboard
npm install
```

### 運行開發服務器

```bash
npm run dev
```

應用將在 `http://localhost:3000` 啟動並自動打開瀏覽器。

### 構建生產版本

```bash
npm run build
```

構建產物將生成在 `dist` 目錄。

### 預覽生產版本

```bash
npm run preview
```

## 🎨 設計特點

### 顏色主題
- **主背景**: `#0a0a0a` - 深黑色
- **次背景**: `#1a1a1a` - 淺黑色
- **強調色**: `#00ff88` (綠色), `#00ccff` (藍色)
- **文字**: 白色與灰色層次

### 互動效果
- 平滑的過渡動畫
- 懸停效果提升用戶體驗
- 響應式卡片佈局
- 動態數據可視化

## 📈 實驗數據概覽

實驗涵蓋：
- 3 輪測試 (Round 1-3)
- 4 種攻擊模式
- 5 種密碼長度 (8-12 位)
- 多種密碼複雜度類型

關鍵發現：
- 密碼長度每增加 1 位，破解時間增加 26-95 倍
- 字典攻擊比暴力破解快 30-90%
- 優化參數可提升效能 1.5-2 倍
